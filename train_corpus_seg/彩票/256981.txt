德国 公布 大 名单 ： 绝对 核心 伤愈 归队 勒沃 红人 入选 
 　 　 北京 时间 周五 ， 德国队 主教练 勒夫 公布 了 下周 参加 世界杯 预选赛 的 22 人大 名单 。 德国队 将 在 10 月 11 日 以及 10 月 15 日 对战 俄罗斯 与 威尔士 。 
 　 　 因伤 错过 前 两场 与 列支敦士登 与 芬兰 比赛 的 巴拉克 以及 弗林斯 重返 勒夫 的 名单 。 弗雷德 里 希 、 沙尔克 04 的 琼斯 、 替补 门将 阿德勒 也 入选 了 本次 国家队 阵容 。 另外 ， 勒沃库森 的 赫尔姆斯 凭借 最近 出色 的 表现 ( 6 场 比赛 攻入 7 球 ) 也 得以 入选 。 梅策 尔德 、 小将 马林 没有 入选 。 
 　 　 附 德国队 大 名单 ： 
 　 　 门将 ： 恩克 ( 汉诺威 96 ) 、 维泽 ( 不莱梅 ) 、 阿德勒 ( 勒沃库森 ) 
 　 　 后卫 ： 弗里茨 ( 不莱梅 ) 、 弗雷德 里 希 ( 柏林 赫塔 ) 、 M - 扬森 ( 汉堡 ) 、 拉姆 ( 拜仁 慕尼黑 ) 、 默特 萨克 ( 不来梅 ) 、 塔什 彻 ( 斯图加特 ) 、 韦斯特 曼 ( 沙尔克 04 ) 
 　 　 中场 ： 巴拉克 ( 切尔西 ) 、 弗林斯 ( 不莱梅 ) 、 希策尔 斯贝格 ( 斯图加特 ) 、 琼斯 ( 沙尔克 04 ) 、 罗尔费斯 ( 勒沃库森 ) 、 施魏 因 施泰格 ( 拜仁 慕尼黑 ) 、 特罗 霍夫斯 基 ( 汉堡 ) 
 　 　 前锋 ： 戈麦斯 ( 斯图加特 ) 、 赫尔姆斯 ( 勒沃库森 ) 、 克洛泽 ( 拜仁 慕尼黑 ) 、 库兰伊 ( 沙尔克 04 ) 、 波 多尔斯 基 ( 拜仁 慕尼黑 )
